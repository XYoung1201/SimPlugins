# Linear Fitting Library

## 项目介绍

此库提供了一个线性拟合类，用于根据输入的数据文件进行线性拟合并进行预测。该项目使用了Eigen库进行矩阵操作，并提供了一种方便的方法来进行线性预测。

## 依赖

- C++11 或更高版本
- Eigen库

## 主要功能

### 文件读取

从指定的CSV文件中读取数据，并解析成矩阵形式。文件格式应为：

- 第一行：特征数量 `m` 和输出值数量 `n`。
- 后续行：每行包括 `m` 个输入特征和 `n` 个输出值。

### 线性拟合

使用读取的数据进行线性拟合，计算出权重矩阵 `W`。

### 预测

使用拟合后的模型进行预测，输入特征向量，输出预测结果。

## 安装

1. 克隆此仓库。
2. 确保您的系统已安装Eigen库。
3. 使用适当的编译器编译项目。

## 使用方法

### 创建对象

使用文件路径创建`LinearFitting`对象，该文件应包含所需的线性拟合数据。

```cpp
LinearFitting fitting("path/to/yourfile.txt");
```

### 预测

使用`Predicting`方法预测新的输入值。

```cpp
double x[] = {value1, value2, ...};
double* y = fitting.Predicting(x);
```

## 代码结构

- `LinearFitting`: 主类，提供线性拟合和预测功能。
- `getW`: 私有方法，用于计算权重矩阵。
- `Predicting`: 公开方法，用于进行预测。
